* mesa-micro
** Purpose
This is some boiler-plate code to do simple calculations using the
microphysics in MESA.  It is not particularly general, but has been
handy for me.
** Usage
#+BEGIN_EXAMPLE
make
./micro
#+END_EXAMPLE
** MESA version
This currently works with r10398.
